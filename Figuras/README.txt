 Participante: Gonzalo M. Torres Medina
 email: gontamx@yahoo.com.mx
 Descripcion: Programa en Java que pide el valor de un lado y calcula el área y perímetro de un cuadrado, y el perímetro y el volumen de un cubo.
              El programa hace uso de la herencia al tener una clase llamada Cuadrado y una clase llamada Cubo que hereda de cuadrado.
 Instrucciones: El usuario introduce un número como valor de un lado y se le presenta el siguiente menú:
               1) Calcular el área de un cuadrado.
 			   2) Calcular el perímetro de un cuadrado.
 			   3) Calcular el perímetro de un cubo.
               4) Calcular el volumen de un cubo.
 			   0) Fin del programa.
